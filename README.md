
基于 **OpenCV** 和 **深度学习模型** 的实时人脸识别系统。  
系统通过摄像头采集视频流，使用 `yunet.onnx` 模型进行人脸检测，提取人脸区域后使用 `face_recognizer_fast.onnx` 模型生成 128 维特征向量，并与本地人脸数据库进行余弦相似度比对，实现多用户身份识别。  

## ✨ 功能特性
- 实时视频流人脸检测与识别
- 支持人脸注册与身份验证
- 使用 ONNX 模型 (`yunet.onnx` + `face_recognizer_fast.onnx`)
- 余弦相似度比对，识别准确率 >90%
- 实时显示人脸框与识别结果
- 支持多张人脸同时检测与标注
- 可保存当前帧图像用于调试或数据扩展
